1.【单选题】在 Module 7 中，我们引入了 Florence-2 模型。关于这个模型，以下描述正确的是？（C）
A. 它是一个纯语言模型（LLM），只能处理文字
B. 它是一个语音合成模型（TTS），只能说话
C. 它是一个视觉-语言模型（VLM），既懂视觉也懂语言
D. 它只能用于生成图片（类似于 Midjourney）
答案：C
难易程度：易
答案解析：Florence-2 是 Microsoft 发布的 VLM (Vision-Language Model)，它的核心能力是“看图说话”，能够处理图像输入并生成对应的文本描述（Caption）、检测框（Detection）或 OCR 结果。
知识点：模型基础；VLM
课程目标：能力目标（10分）

2.【单选题】在 Lesson 26 中，我们使用 SiliconFlow 提供的 DeepSeek API。关于 API 中的 `temperature` 参数，以下说法正确的是？（B）
A. 它控制 API 的响应速度，数值越大速度越快
B. 它控制生成内容的随机性（创造力），数值越大越“发散”
C. 它控制服务器的硬件温度，防止过热
D. 它控制生成文本的长度
答案：B
难易程度：中
答案解析：Temperature（温度）是采样参数。温度低（如 0.1）时，模型倾向于选择概率最高的词，输出理性、稳定；温度高（如 0.9）时，模型会尝试概率较低的词，输出更有创意但也更容易出错。
知识点：API参数；Prompt工程
课程目标：能力目标（10分）

3.【单选题】我们将“视觉模型 (Florence-2)” + “逻辑模型 (DeepSeek)” + “语音模型 (CosyVoice)” 串联起来工作的模式，被称为什么？（A）

知识点：系统架构；Pipeline
课程目标：能力目标（10分）

7.【单选题】关于“结构化输出 (Structured Output)”，为什么我们在开发应用时更喜欢 JSON 而不是普通文本？（A）
A. JSON 方便程序解析和提取特定字段（如 `result['score']`）
B. JSON 比普通文本更短
C. JSON 看起来更高级

10.【单选题】关于 System Prompt（系统提示词）和 User Prompt（用户提示词），以下说法正确的是？（A）
