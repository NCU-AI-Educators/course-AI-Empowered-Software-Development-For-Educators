1.【单选题】在 Web 开发的“餐厅隐喻”中，如果**浏览器 (Browser)** 是顾客，**API 接口** 是菜单，那么 **FastAPI 服务器** 扮演的角色是？（C）

B. 负责结账的收银台

知识点：Web架构；C/S模型

2.【单选题】HTTP 协议定义了多种“动词”来表达操作意图。如果我们想要**获取**当前的菜单列表，不改变服务器上的任何数据，应该使用哪种请求方法？（A）
A. GET
B. POST
C. DELETE
D. PUT
答案：A
难易程度：易
答案解析：GET 请求用于“获取”资源，是“安全”且“幂等”的（只读）。POST 通常用于“提交”数据（如注册、下单），会改变服务器状态。
知识点：HTTP协议；RESTful风格
课程目标：能力目标（10分）

3.【单选题】在 Python 的 FastAPI 代码中，装饰器 `@app.get("/menu")` 的主要作用是？（B）
A. 启动服务器
B. 定义路由 (Routing)：告诉服务器当用户访问 `/menu` 时执行哪个函数
C. 定义数据库结构
D. 自动生成前端 HTML 页面
答案：B
难易程度：中
答案解析：装饰器 (Decorator) `@app.get(...)` 的作用是将一个 URL 路径（路由）映射到一个 Python 处理函数上。当请求到达该路径时，FastAPI 会自动调用对应的函数。
知识点：FastAPI基础；路由
课程目标：能力目标（10分）

4.【单选题】Web 前后端交互中最常用的数据格式是 **JSON**。关于 JSON，以下说法错误的是？（D）
A. 它是一种轻量级的文本数据交换格式
B. 它的语法结构与 Python 的字典 (Dictionary) 非常相似
C. 它易于人阅读，也易于机器解析
D. 它只能用于 Python 语言，其他语言无法识别
答案：D
难易程度：易
答案解析：JSON (JavaScript Object Notation) 是语言无关的（Language Independent）。虽然它源自 JS，且长得像 Python 字典，但几乎所有编程语言（Java, C++, Go等）都支持 JSON。它是互联网的“通用语”。
知识点：数据格式；JSON
课程目标：能力目标（10分）

5.【单选题】前端 JavaScript 代码中使用 `fetch()` 指令向服务器请求数据时，通常会使用 `.then()` 语法。这体现了 Web 编程的什么特性？（C）
A. 同步 (Synchronous)：必须等数据回来才能做别的事
B. 面向对象 (Object-Oriented)
C. 异步 (Asynchronous)：发出请求后不等待，数据回来后再处理
D. 静态 (Static)
答案：C
难易程度：中
答案解析：网络请求需要时间。为了不让浏览器“卡死”（假死），`fetch` 采用异步非阻塞模式。`.then()` 的含义是“等数据回来了，**然后**再执行后续操作”，在此期间用户依然可以滚动页面。
知识点：前端交互；异步编程
课程目标：能力目标（10分）

6.【单选题】当你的网页 (运行在 5500 端口) 试图访问后端的 API (运行在 8000 端口) 时，浏览器控制台报错 `CORS error`。这是因为？（B）
A. 服务器的代码写错了，崩溃了
B. 浏览器的“同源策略”安全机制拦截了跨域请求
C. 电脑断网了
D. 你的 Python 版本太低
答案：B
难易程度：中
答案解析：CORS (跨域资源共享) 错误是浏览器的安全机制，默认禁止不同端口/域名的网站互相访问，以防恶意窃取数据。解决方法是在后端添加 CORS 中间件（Middleware）来“发放通行证”。
知识点：Web安全；CORS
课程目标：能力目标（10分）

7.【单选题】我们将 MUD 游戏从“命令行脚本”重构为“Web 应用”时，程序的核心运行模式发生了什么变化？（A）
A. 从“死循环 (While True)” 变为 “事件驱动 (Event-Driven)”

课程目标：能力目标（10分）

5.【多选题】RESTful API 设计风格提倡“资源导向”。以下 URL 设计中，符合 RESTful 风格的有哪些？（AD）
A. `POST /orders` (创建一个新订单)
B. `GET /get_all_orders_please` (动词放在 URL 里了)
C. `POST /update_order_price` (动词放在 URL 里了)
D. `GET /orders/101` (获取 ID 为 101 的订单详情)
答案：AD
难易程度：难
答案解析：RESTful 风格主张 URL 只包含**名词**（资源，如 `/orders`），而操作动作由 **HTTP Method**（动词，如 GET/POST）来表达。B 和 C 把动词写在 URL 里，是典型的 RPC 风格，不符合 RESTful 规范。
知识点：API设计；RESTful
课程目标：创新目标（10分）
