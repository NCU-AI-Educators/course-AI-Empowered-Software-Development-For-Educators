# 实践挑战：创造你的第二个“魔法”

## 目标：动态粒子星空

你可以尝试使用下面的**中文**或**英文**指令，看看AI是否都能理解！

### 中文指令
```
请创建一个名为 particles.html 的独立HTML文件，其中包含一个全屏的画布。画布上要有浮动的、发光的粒子动画，当粒子彼此靠近时，它们之间会用线条连接。粒子动画还需要对鼠标的移动做出反应。
```

### English Prompt
```
Create a single HTML file named particles.html. It should feature a full-screen canvas with an animated background of floating, glowing particles that connect with lines when they are close to each other. The particles should also react to mouse movement.
```
