# 实践挑战：创造你的第一个“魔法”

## 目标：交互式3D地球

你可以尝试使用下面的**中文**或**英文**指令，看看AI是否都能理解！

### 中文指令
```
请创建一个名为 globe.html 的独立HTML文件，使用Three.js渲染一个3D地球。这个地球必须有逼真的、全彩色的地球纹理，可以交互（通过鼠标旋转和缩放），并且背景中要带有星星。地球本身也需要缓慢地自转。
```

### English Prompt
```
Create a single HTML file named globe.html. This file should render a 3D globe using Three.js with no external dependencies. The globe must have a realistic, full-color texture of the Earth, be interactive (rotatable and zoomable with the mouse), and have stars in the background. The globe should also slowly rotate on its own.
```
