## 使用说明

### 转换为PPT
使用Marp工具将Markdown文件转换为PPTX格式：
```bash
# 安装marp-cli
npm install -g @marp-team/marp-cli

# 转换单个文件
marp module1_lesson1.md -o module1_lesson1.pptx

# 批量转换
marp *.md
```

### 转换为PDF
```bash
# 转换为PDF格式
marp module1_lesson1.md -o module1_lesson1.pdf
```

### 在VS Code中预览
1. 安装Marp VS Code扩展
2. 打开.md文件
3. 按Ctrl+Shift+V或点击右上角的Marp图标预览

## 文件格式说明

每个讲义文件都遵循Marp格式规范：
- 使用YAML front matter定义幻灯片属性
- 每页幻灯片以`---`分隔
- 支持Markdown语法
- 内嵌CSS样式控制外观
