# 题目导入模板 (Question Import Template)

在智慧课堂助手中导入题目时，请复制并参考以下 Markdown 格式。

## 基本规则
1. 使用 `##` 开头标记题目内容（支持单行或多行）。
2. 使用 `- [ ]` 标记选项。
3. 在中括号内填入 `x` 或 `X` (即 `- [x]`) 标记为正确答案。
4. 系统会自动根据正确答案的数量判断是单选还是多选。
5. 题目导入时会自动去除标题中的序号（如 `1.`），系统将重新编号。

---

## 示例 1: 单选题 (Single Choice)

```markdown
## 1. 这是第一道单选题
- [ ] A. 错误选项
- [x] B. 正确选项 (后面也可以跟文字)
- [ ] C. 错误选项
- [ ] D. 错误选项
```

## 示例 2: 多选题 (Multiple Choice)
只要有两个及以上正确选项，系统自动识别为多选。

```markdown
## 2. 下列哪些是编程语言？
- [x] A. Python
- [x] B. Java
- [ ] C. HTML (标记语言)
- [x] D. C++
```

## 示例 3: 多行题目 (Multi-line Question)
题目内容可以跨越多行，直到遇到第一个选项为止。

```markdown
## 3. 阅读理解题
这是一道复杂的题目。
题目描述占据了多行。
请根据上下文选择最合适的答案。
- [ ] A. 选项 1
- [x] B. 选项 2
- [ ] C. 选项 3
```
