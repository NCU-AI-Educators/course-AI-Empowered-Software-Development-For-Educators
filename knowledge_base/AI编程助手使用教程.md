# AI编程助手使用教程 (Qwen-Code版)

## 1. 简介

欢迎使用Qwen Code!

你可以把它想象成一位住在你的电脑里的编程专家. 它是一个强大的**命令行AI助手**, 专门为编程任务优化. 在本课程中, 我们将使用它来帮助我们理解代码, 生成代码, 修改代码, 甚至自动化一些繁琐的工作.

**我们为什么选择它?**

*   **功能强大**: 它背后由强大的Qwen3-Coder系列模型驱动, 对代码的理解和生成能力非常出色.
*   **操作简单**: 它继承了Google Gemini CLI的设计, 可以通过类似聊天的方式在命令行里与AI进行交互.
*   **免费使用**: 它提供了非常慷慨的免费额度 (每天2000次请求), 完全能满足我们课程学习和后续探索的需求.

## 2. 安装与配置 (三步走)

请完全按照以下步骤操作, 这是最简单快捷的配置方式.

### 第一步: 安装 Node.js (编程环境)

Qwen Code需要Node.js环境才能运行. 如果你的电脑还没有安装, 请按以下步骤操作:

1.  访问 [Node.js官方网站](https://nodejs.org/en/download).
2.  选择 **LTS (长期支持版)**, 根据你的操作系统 (Windows或macOS) 下载安装包.
3.  双击安装包, 按照提示一路点击"下一步"完成安装.

### 第二步: 安装 Qwen Code

打开你电脑的**终端 (Terminal)**:
*   **macOS**: 在"应用程序" -> "实用工具"里找到"终端(Terminal)".
*   **Windows**: 在开始菜单搜索 "PowerShell" 或 "CMD" 并打开.

在终端里, 根据你的操作系统, **执行以下两行命令中的任意一行**:

*   **如果你是 macOS 用户 (推荐使用Homebrew):**
    ```bash
    brew install qwen-code
    ```
*   **如果你是 Windows 用户, 或macOS未使用Homebrew:**
    ```bash
    npm install -g @qwen-code/qwen-code@latest
    ```

安装完成后, 可以输入 `qwen --version` 来验证是否安装成功. 如果出现版本号, 说明已成功.

### 第三步: 首次运行与授权

这是最关键也最简单的一步. 只需在终端里输入以下命令:

```bash
qwen
```

1.  程序会**自动打开你的浏览器**, 跳转到千问(Qwen)的登录授权页面.
2.  请使用你的个人账户**登录并点击"同意授权"**. (如果没有账号, 可以免费注册一个).
3.  授权成功后, 浏览器会提示"登录成功", 此时你就可以关闭浏览器页面, 回到终端了.

终端界面会显示欢迎信息, 并出现一个 `>` 提示符. 这意味着你已经成功连接到AI助手, 可以开始对话了!

## 3. 基本使用方法

进入 `qwen` 的交互界面后, 你就可以像微信聊天一样和AI对话了.

*   **直接提问**:
    ```
    > 用python写一个计算阶乘的函数
    ```
*   **引用文件**: 这是最常用的功能! 使用 `@` 符号可以引用当前目录下的文件, 让AI读取并理解它.
    ```
    > @course_outline_40min.md 总结一下这个课程大纲的模块五主要讲了什么
    ```
    ```
    > @my_script.py 这段代码有什么问题?
    ```
*   **让AI执行命令**: 你可以让AI帮你执行终端命令, 比如运行脚本, 查看文件等.
    ```
    > 帮我运行 @my_script.py 这个脚本
    ```
*   **退出会话**:
    *   输入 `/exit` 或 `/quit`.
    *   在提示符为空时, 按 `Ctrl + D`.

## 4. 核心命令 (会话管理)

在 `>` 提示符后, 输入以下命令可以管理你的对话:

*   `/help`: 显示所有可用的命令.
*   `/clear`: 清空当前的对话历史, 开始一个全新的对话. (当你觉得AI开始"糊涂"时很有用).
*   `/stats`: 查看当前会话的token(令牌)使用情况.
*   `/history`: 查看你和AI的完整对话历史.

## 5. 课程实践场景举例

*   **启动项目时**:
    `> 我想写一个Python脚本, 用来分析Excel里的学生成绩, 请帮我生成一个基本的代码框架, 需要用到pandas库.`
*   **代码写不下去时**:
    `> @my_script.py 我已经读取了Excel, 接下来如何计算每一科的平均分?`
*   **遇到报错时**:
    `> @my_script.py 我运行这个脚本时, 出现了 'FileNotFoundError' 错误, 请帮我分析原因并修复它.`
*   **学习新知识时**:
    `> @my_script.py 请逐行解释这段代码的含义.`

## 6. (选学) 高级配置: 使用其他模型API

Qwen Code同样支持配置其他兼容OpenAI格式的API, 例如ModelScope的免费API. 这部分为选学内容, 感兴趣的学员可以自行研究.

你可以在你的项目目录下创建一个名为 `.env` 的文件, 并填入以下内容:

```
# 示例: 使用ModelScope的免费API
OPENAI_API_KEY="你的ModelScope API Key"
OPENAI_BASE_URL="https://api-inference.modelscope.cn/v1"
OPENAI_MODEL="Qwen/Qwen3-Coder-480B-A35B-Instruct"
```

配置后, 在该目录下运行 `qwen`, 将会使用你指定的模型.
