### 私人专家提示模板库 (Private Expert Prompt Template Library)

#### **引言**
欢迎使用本模板库！这里的每一个模板，都是为了将您从“AI的使用者”提升为“AI的指挥家”。请不要直接复制粘贴，最好的方式是理解其背后的“习惯”思想，然后根据您的具体场景，修改和组合这些模板。

---

### **习惯1：热身教练 (Warm-up Coach)**
*   **目标**：消除“冷启动”摩擦，快速进入工作状态。

#### **模板1.1：每日任务规划师**
```
我今天感觉有些不知所措，这是我脑子里所有的任务和想法：

[在此处粘贴你的任务、想法、担忧列表，内容越原始、越混乱越好]

请帮我：
1. 将这些内容按项目或类别（如：工作、学习、私人事务）进行分类整理。
2. 根据重要性和紧急性，识别出今天最重要的三件事（Top 3）。
3. 以清晰的待办事项列表形式呈现，并用一句话建议我应该从哪一件开始，为什么。
```
*   **专业提示**：每天早上开始工作前，花5分钟进行一次“脑力转储”，这是开启高效一天的最佳仪式。

#### **模板1.2：会议准备助手**
```
我即将与 [人物姓名/公司名称] 开会，会议主题是 [会议主题]。

这是我目前掌握的关于对方的背景资料：[粘贴对方的公开简介、公司网站、过往邮件、相关新闻等]

请帮我：
1. 总结此人/公司的核心要点（如：近期成就、核心业务、个人风格等）。
2. 基于我们的会议主题，预测对方可能最关心的3个问题。
3. 为我设计3个有深度、能体现我已做好功课的提问。
```
*   **专业提示**：提供给AI的背景资料越丰富，它为你准备的“弹药”就越精良。

### **习惯2：分解微任务 (Decompose Micro-tasks)**
*   **目标**：将复杂、模糊的大任务，拆解为清晰、可执行的小步骤。

#### **模板2.1：项目拆解专家**
```
我需要完成一个复杂的任务：[描述你的宏大任务，例如：策划一个为期三个月的线上营销活动]。

请扮演一个拥有10年经验的顶级项目经理，将这个宏大任务分解为5-7个清晰、可执行、有先后顺序的关键阶段。

然后，请仅先详细阐述“第一阶段”需要具体执行的3-5个步骤。
```
*   **专业提示**：这个方法的核心是“逐层深入”。让AI先搭好框架，再聚焦到第一步，可以有效避免AI在长篇输出中“跑偏”或“遗忘”。

### **习惯3：利用初稿 (Leverage Drafts)**
*   **目标**：用最快的速度生成一个“60分”的初稿，克服“空白页恐惧症”。

#### **模板3.1：快速草稿生成器**
```
我需要写一篇关于 [主题] 的 [文章/邮件/报告/演讲稿]。

请不要追求完美，为我快速生成一个包含“引言、三个主要论点、结论”这几个基本部分的“60分”初稿。

我的目标是得到一个可以立刻开始动手修改和迭代的基础，而不是一篇完美的成品。请开始吧。
```
*   **专业提示**：明确告诉AI你“不要完美”，可以降低它的“思考”负担，从而更快地得到结果。

---

### **习惯4：递归自我完善 (Recursive Self-Improvement)**
*   **目标**：让AI自我批判，不断迭代，将一份60分的文稿提升到90分。

#### **模板4.1：“批评-改进”循环**
```
我将给你一段我的初稿。请严格遵循以下“批评-改进”流程，循环两次：

1.  **作为一名 [指定角色，如：挑剔的《经济学人》编辑 / 资深营销总监 / 追求极致用户体验的产品经理]，用专业的眼光批判性地评估这段文字，从 [指定维度，如：逻辑清晰度、语言说服力、情感吸引力] 等方面，明确指出3个最主要的弱点。**
2.  **基于你发现的弱点，生成一个“优化版本”。**

请在两次循环后，提交最终的、经过两轮打磨的完善版本。

这是我的初稿：“[在此处粘贴你的文本]”
```
*   **专业提示**：为AI指定一个“批判者”角色，并明确“批判维度”，是这个技巧成功的关键。

---

### **习惯5：“展示”，而不仅仅是“告知” (Show, Don't Tell)**
*   **目标**：让AI精准模仿你想要的风格、格式或语气。

#### **模板5.1：风格模仿大师**
```
我需要你模仿我提供的范本，创作一段新内容。请严格学习范本的 [指定风格元素，如：语气、句式结构、段落长度、表情符号用法、标签格式等]。

**范本1：** “[粘贴范本1]”
**范本2：** “[粘贴范本2]”

**我的新任务：** 现在，请用完全相同的风格，为 [你的新主题] 创作一段内容。
```
*   **专业提示**：提供的范本质量越高、风格越统一，AI的模仿效果就越好。建立你自己的“范本库”是成为AI高手的必经之路。

### **习惯6：镜中分析师 (Analyst in the Mirror)**
*   **目标**：利用AI挑战自己的思维定势，发现计划中的盲点和漏洞。

#### **模板6.1：压力测试专家**
```
我有一个计划/想法，现在需要你扮演一个 [指定角色，如：经验丰富但极度保守的财务审计师 / 对用户体验要求苛刻的“刺头”用户 / 永远在寻找更好方案的竞争对手]。

请对我的计划进行一次彻底的压力测试，提出3个最尖锐、最可能被我忽略的潜在风险、反驳论点或更优的替代方案。

这是我的计划：“[在此处粘贴你的计划或想法]”
```
*   **专业提示**：赋予AI一个“对立面”的角色，能最大限度地激发它提出颠覆性观点的能力。

---

### **习惯7：思维向导 (Thinking Guide)**
*   **目标**：当你想法混乱时，利用AI的提问能力，引导自己理清思路。

#### **模板7.1：苏格拉底式提问者**
```
我的脑子现在很乱，但我有一个关于 [描述你模糊的想法，例如：为我的咖啡店做一个新的会员体系] 的模糊想法。

我现在无法给你清晰的指令。请不要直接给我方案。请像一位循循善诱的思维向导（或苏格拉底），通过向我连续提问的方式，一步步引导我理清我的思路。

请先提出第一个问题，帮助我明确我的最终目标是什么。
```
*   **专业提示**：这个技巧的关键是禁止AI直接给答案，强制它进入“提问模式”。这是将AI从“工具”变为“教练”的终极用法。
